Documentación del Sistema

Arquitectura modular con sistema de rutas avanzado:

  • Router.js Maneja navegación entre vistas
  • LoadingScreen.js Pantalla de carga con progreso
  • Textures.js Genera texturas planetarias
  • Objects.js Crea objetos 3D (planetas, estrellas)
  • SolarSystem.js Configura sistema solar
  • Animation.js Controla animaciones y transiciones
  • PlanetData.js Datos de planetas y rutas

Haz clic en los planetas o usa el sistema de rutas superior para navegar.
