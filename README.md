# 🌌 Sistema Solar Modular

Un sistema interactivo del **Sistema Solar** desarrollado con **HTML, CSS y JavaScript**.  
Incluye cámara de seguimiento, sombras dinámicas, soporte para texturas y una **consola de comandos** para controlar la navegación entre planetas.  
Diseñado de manera **modular** para facilitar su integración en frameworks o librerías (React, Vue, Angular, etc.).

---

## 🚀 Características

- 🌍 Visualización planeta por planeta con cámara de seguimiento.  
- 💻 Consola de navegación interactiva con comandos personalizados.  
- 🎨 Soporte para efectos visuales: sombras y texturas.  
- ⚡ Arquitectura modular en **JavaScript** para fácil escalabilidad.  
- 🛰️ Base exportable a cualquier framework o librería.

## 📂 Árbol de directorios

```plaintext
sistema-solar-modular/
│── css/
│   └── style.css                # Estilos principales
│
│── js/
│   ├── modules/
│   │   ├── Animation.js         # Control de animaciones
│   │   ├── Console.js           # Lógica de la consola de comandos
│   │   ├── LoadingScreen.js     # Pantalla de carga
│   │   ├── Objects.js           # Definición de objetos 3D
│   │   ├── PlanetData.js        # Datos de los planetas
│   │   ├── Router.js            # Sistema de navegación entre planetas
│   │   ├── SolarSystem.js       # Configuración del sistema solar
│   │   ├── Textures.js          # Manejo de texturas
│   │   └── Utils.js             # Funciones utilitarias
│   │
│   └── main.js                  # Punto de entrada principal
│
├── index.html                   # Archivo principal del proyecto
└── README.md                    # Documentación del proyecto
